## 用户-图书评分矩阵构建

通过 `build_user_item_matrix()` 函数构建用户-图书评分矩阵，其中用户对图书的评分基于以下行为：

- 收藏书籍：评分=1
- 心愿书籍：评分=0.5
- 其他书籍：评分=0

## 计算用户相似度

使用余弦相似度 (`cosine_similarity`) 计算用户之间的相似度。

## 推荐逻辑

- 根据用户相似度矩阵，为目标用户推荐与其行为相似的其他用户喜欢的书籍。
- 推荐书籍为目标用户未曾收藏或添加至心愿单的。
- 推荐书籍数量上限为12本。

## 协同过滤

- 基于用户的协同过滤：通过计算用户间的相似度，推荐相似用户喜欢的图书。

此推荐系统通过以上机制，为用户提供精准的图书推荐服务。
